# INPUT
Node.js project with Express.js app running on port 3000

# PROCESS
1. Start the Express.js server by running `node app.js`
2. The server should be running and listening on port 3000
3. Verify the server is running by checking if it responds to requests

# OUTPUT
- The Express.js server running on port 3000
- Server responding to GET /todos and POST /todos endpoints
- Server process remains active in the terminal
