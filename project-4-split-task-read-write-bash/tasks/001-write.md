# INPUT
Node.js project document for a todo app

# PROCESS
1. Create a basic Express.js application in app.js
2. Set up a todos collection in a JSON file as the database
3. Create a simple GET /todos endpoint to fetch all todos
4. Create a simple POST /todos endpoint to add a new todo

# OUTPUT
- app.js: Express.js server with CRUD endpoints for todos
- todos.json: JSON file to store todo items
- A simple todo API with GET /todos and POST /todos endpoints
