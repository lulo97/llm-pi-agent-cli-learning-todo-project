# Task 5: Setup E2E Tests

Goal: Create the testing structure and initial test files.

Requirements:
- Create a `tests` directory.
- Create four JavaScript test files in the `tests` directory:
    - `add_todo.spec.js`
    - `update_todo.spec.js`
    - `delete_todo.spec.js`
    - `read_todo.spec.js`
- Each test file should be set up to use Playwright and target the `data-test-id` attributes from Task 4.
