# Task 2: Implement CRUD API Endpoints

Goal: Implement full CRUD API endpoints for a ToDo entity.

Requirements:
- Use `todo.json` as the database.
- Implement endpoints for:
    - Create (POST) a new ToDo.
    - Read (GET) all ToDos.
    - Read (GET) a specific ToDo by ID.
    - Update (PUT/PATCH) an existing ToDo.
    - Delete (DELETE) a ToDo.
- Data structure in `todo.json` should support at least an ID, title, and description.
